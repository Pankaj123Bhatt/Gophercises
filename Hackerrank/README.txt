This repository contains solution to 2 hackerrank problems.

1. CamelCase : It is a concatenation of one or more words consisting of English letters. All letters in the first word are lowercase. For each of the subsequent words, the first letter is uppercase and the rest are lower. We need to find the total no.of words in the sequence.

2. CaeserCipher : Its a program to encrypt a string by rotating all its characters by k rotations.
